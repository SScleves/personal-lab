# k6

Grafana's load-testing tool — the "player swarm" generator for [[07 Demo Script]] step 2.

## Verified 2026-07-06

- OSS (AGPL-3.0), free; native **arm64** binaries and Docker images — runs on the
  [[OCI Always Free]] cluster or the laptop.
- Protocols: HTTP, **WebSockets**, gRPC, browser. **No UDP in core** (xk6-udp extensions exist but
  are tiny community projects — avoid).
- [[Grafana Cloud]] free tier includes **500 k6 cloud VUh/month** — stream local-run results there
  for pretty on-camera dashboards.

## Consequences for the lab

- The agar.io-clone fleet (WebSocket — [[Game Servers]]) is k6-testable natively:
  ramping VUs that connect, move, disconnect = FleetAutoscaler trigger + latency SLO load.
- Minecraft is NOT k6 territory (raw UDP/TCP game protocol) — use a Minecraft bot library;
  k6 can still probe its TCP status port for a liveness check.
- Also load-test the [[Gemini Chatbot]] HTTP endpoint (gently — free-tier RPM limits, [[Free Tier Limits]]).
